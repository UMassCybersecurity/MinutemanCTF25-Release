# MinutemanCTF25-Release 

MinutemanCTF25 was the second annual competition hosted by the UMass Cybersecurity Club. The CTF ran for a period of 36 hours from 6PM Oct 17 to 12PM Oct 19 (ETC). In total, there were 100+ players and 50+ teams took part in this competition. Challenges featured all categories like crypto, rev, web, pwn, osint, misc, forensics, hardware and rf.
